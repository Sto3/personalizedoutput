## Testing Strategy

### Phase 1: Initial Test (1 week each)
1. Start with Version A for all products
2. Track: Views, Favorites, Conversion Rate, Revenue

### Phase 2: Swap Test
1. If Version A underperforms expectations, switch to Version B
2. Compare metrics week-over-week

### Phase 3: Seasonal Adjustment
- Nov 15 - Dec 24: Use gift-focused versions (B or E)
- Dec 26 - Jan 15: Use New Year/goals versions (A or C)
- Jan 16+: Use transformation/outcome versions (C or D)

### Key Metrics to Track
- **Click-through rate** (views from search)
- **Conversion rate** (purchases / views)
- **Favorite rate** (indicates interest)
- **Revenue per listing**

### Etsy SEO Tips
1. Put most important keywords first (Etsy weights first 40 chars heavily)
2. Include price point keywords: "personalized", "custom", "gift"
3. Seasonal keywords boost during holidays
4. "Digital download" and "PDF" signal instant delivery
5. Include who it's for: "for her", "for him", "for kids"
